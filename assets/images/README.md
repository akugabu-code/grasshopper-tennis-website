# 画像ファイルについて

このディレクトリには、ウェブサイトで使用する画像ファイルを配置してください。

## フォルダ構成

```
assets/images/
├── events/          # 年間行事の写真（月別）
├── tournaments/     # 大会写真（すべての大会）
└── members/         # メンバー紹介写真
```

## 必要な画像ファイル

- `apr_compa_3.jpg`（追加する場合）

- `jun_spring_trip.jpg`

#### 7月 - 夏季個人杯

- `jul_individual_2.jpg`
- ...

#### 8月 - ヨネックス杯＆夏合宿
- `aug_yonex_1.jpg`
- `aug_summer_camp.jpg`
- ...

#### 9月 - 秋合宿

**推奨サイズ**: 1200x900px（4:3比率）
**ファイル形式**: JPG または WebP
**ファイルサイズ**: 500KB以下（圧縮推奨）

### 2. 大会写真（tournaments/フォルダ）

すべての大会の写真を番号順に配置してください：

- `tournament1.jpg`
- `tournament2.jpg`
- `tournament3.jpg`
- `tournament4.jpg`
- `tournament5.jpg`
- `tournament6.jpg`
- ...（必要な枚数だけ追加）

**推奨サイズ**: 1200x900px（4:3比率）
**ファイル形式**: JPG または WebP
**ファイルサイズ**: 500KB以下（圧縮推奨）

### 3. メンバー写真（members/フォルダ）

幹部メンバーの写真を以下のファイル名で配置してください：

- `member1.jpg` - 代表
- `member2.jpg` - 副代表
- `member3.jpg` - 会計
- `member4.jpg` - 広報

**推奨サイズ**: 600x600px（正方形）
**ファイル形式**: JPG または WebP
**ファイルサイズ**: 200KB以下（圧縮推奨）

## 画像の準備方法

### 1. 写真を撮影・収集
- 活動の様子がわかる写真
- 明るく、ピントが合った写真
- メンバーの表情が見える写真

### 2. 画像の編集（任意）
- トリミングして構図を整える
- 明るさやコントラストを調整
- 不要な背景をぼかす

### 3. 画像の圧縮
ファイルサイズを小さくすることで、ページの読み込み速度が向上します。

**無料圧縮ツール**:
- [TinyPNG](https://tinypng.com/) - オンラインで簡単圧縮
- [Squoosh](https://squoosh.app/) - Googleの画像最適化ツール
- [Compressor.io](https://compressor.io/) - 高品質圧縮

### 4. ファイル名の変更
上記のファイル名に合わせて名前を変更してください。

### 5. このディレクトリに配置
`assets/images/` ディレクトリに画像をコピーしてください。

## ダミー画像の使用（テスト用）

画像を準備するまでの間、以下のダミー画像サービスを使用できます：

### Placehold.co（おすすめ）
```html
<!-- 活動写真用 -->
<img src="https://placehold.co/1200x900/2ecc71/white?text=Activity+Photo" alt="活動写真">

<!-- メンバー写真用 -->
<img src="https://placehold.co/600x600/3498db/white?text=Member+Photo" alt="メンバー写真">
```

### Unsplash（実写のダミー画像）
テニス関連の無料写真:
- https://source.unsplash.com/1200x900/?tennis,sport
- https://source.unsplash.com/600x600/?portrait,person

## 画像の最適化チェックリスト

- [ ] 適切なサイズにリサイズされている
- [ ] ファイルサイズが最適化されている（500KB以下推奨）
- [ ] ファイル名が半角英数字のみ（日本語不可）
- [ ] 正しいファイル形式（JPG、PNG、WebP）
- [ ] 画像の向きが正しい
- [ ] 顔がはっきり見える（メンバー写真）

## 注意事項

### プライバシーに配慮
- 個人が特定できる写真を使用する際は本人の許可を得てください
- 顔をぼかす、後ろ姿にするなどの配慮も検討してください

### 著作権に注意
- 自分で撮影した写真か、使用許可のある写真のみを使用してください
- インターネットから無断で画像をダウンロードして使用しないでください

### ファイル管理
- 元の高解像度画像は別の場所にバックアップしておきましょう
- 定期的に画像を更新して、最新の活動を紹介しましょう

## トラブルシューティング

### 画像が表示されない場合
1. ファイル名が正しいか確認（大文字小文字も区別されます）
2. ファイルが`assets/images/`ディレクトリにあるか確認
3. ファイル形式が対応しているか確認（JPG、PNG、WebP、GIF）
4. ブラウザのキャッシュをクリアして再読み込み

### 画像の読み込みが遅い場合
1. 画像ファイルのサイズを確認（500KB以下が推奨）
2. 画像圧縮ツールで最適化
3. WebP形式への変換を検討

---

**ヒント**: 画像がない場合でも、ダミー画像を使用してレイアウトを確認できます。実際の画像を準備してから差し替えてください。
